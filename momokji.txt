***뭐먹지(momokji)***

개발 취지: 메뉴를 고르지 못하는 미련한 현대 직장인들(나 포함)을 위한 강제 메뉴 선택 어플리케이션

서비스 내용
1. 어플리케이션을 사용하는 유저의 GPS(카카오) 주소를 기반으로 반경 XXm에 있는 메뉴 카테고리를 랜덤으로 하나 추천해주고
(web인 경우 geoLocation이 정확하지 않기때문에 유저가 직접 핀을 꽃을수 있도록~)
2. 해당 카테고리의 가게 목록을 리스트로 보여줌.
3. 가게를 클릭하면 카카오 가게 정보로 연결해서 메뉴를 볼 수 있게 제공
4. 메뉴는 쿨타임 걸어서 바로 가챠 못돌리게 제한. (쿨타임은 5분, 쿨타임 단축 아이템 광고보면 바로 가챠기회 제공)
5. 광고제거 (평생 제거 9,000원, 1달 제거 500원)
6. 랭킹 목록(회원 한정)
	- 메뉴 못고르는 미련한 유저 랭킹(가챠 횟수)
	- 어플리케이션 연속 접속일 수
	- 가게 좋아요 수(만족도가 높다고 판단할 수 있도록)
7. 회원/비회원 닉네임 유무
8. 회원가입은 간편가입으로만 (네이버, 카카오, 구글)
9. 지역 즐겨찾기(지역: 집, 회사)

인터페이스
- 카카오맵 api 호출
- 가게정보를 db에 넣어야되는지 카카오 api에서 뭐가 있는지 확인
- 회원가입 간편가입(insert)
- 가게 좋아요 카운트(update)
- 가챠 기능(select)
- 가챠 카운트(update)
- 앱 접속 카운트(update)
- 지역 즐겨찾기(insert)
- 지역 즐겨찾기(update)
- 지역 즐겨찾기(select)
- 지역 즐겨찾기(delete)
